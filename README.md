# Quiplash Bot для Telegram

Телеграм-бот для игры в стиле Quiplash на русском языке. Соревнуйтесь с друзьями в остроумии!

## Требования

- Python 3.10+
- Токен Telegram-бота (получить у [@BotFather](https://t.me/BotFather))

## Установка

1. Склонируйте или скопируйте проект.

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Откройте файл `config.py` и замените `YOUR_TOKEN_HERE` на токен вашего бота:

```python
BOT_TOKEN = "123456789:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

4. Запустите бота:

```bash
python bot.py
```

## Как играть

1. Каждый игрок пишет боту `/start` в личных сообщениях.
2. Один игрок нажимает "Создать лобби" и получает 6-символьный код.
3. Остальные игроки нажимают "Войти в лобби" и вводят код.
4. Когда набирается 3+ игрока, хост нажимает "Начать игру".

### Раунды

- **Раунд 1 — Смешной ответ**: отвечайте на абсурдные вопросы.
- **Раунд 2 — Дополни фразу**: придумайте концовку к незаконченной фразе.
- **Раунд 3 — Word Remix**: составьте фразу из заданных слов (взятых из ваших ответов в раундах 1-2).

### Голосование

После того как все ответили, начинаются дуэли. Два ответа показываются анонимно, и остальные игроки голосуют за лучший.

### Очки

- **1000 очков** за каждый голос в вашу пользу.
- **2000 бонусных очков**, если ВСЕ проголосовали за вас (Quiplash!).

## Настройки

Все параметры находятся в `config.py`:

| Параметр | Значение | Описание |
|---|---|---|
| `MIN_PLAYERS` | 3 | Минимум игроков для начала |
| `MAX_PLAYERS` | 10 | Максимум игроков в лобби |
| `ANSWER_TIMEOUT` | 90 | Время на ответ (секунды) |
| `VOTE_TIMEOUT` | 30 | Время на голосование (секунды) |
| `SCORE_PER_VOTE` | 1000 | Очков за каждый голос |
| `QUIPLASH_BONUS` | 2000 | Бонус за единогласное голосование |

## Структура проекта

```
quiplash_bot/
├── bot.py              # Точка входа, регистрация обработчиков
├── config.py           # Настройки бота
├── questions.py        # 100 вопросов и фраз
├── requirements.txt    # Зависимости
├── models/             # Модели данных (Player, Lobby, Game, Duel)
├── storage/            # StateManager (in-memory) и Database (SQLite)
├── handlers/           # Обработчики команд, кнопок, сообщений
├── game_logic/         # Паринг, подсчёт очков, управление раундами
└── utils/              # Генерация ID лобби, таймеры
```
